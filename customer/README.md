# Customer service

## API List

Fetch all customer
```bash
[GET] http://localhost:8081/customers 
```
Fetch a customer
```bash
[POST] http://localhost:8081/customers/{customerId}
```
Save customer
```bash
[POST] http://localhost:8081/customers 
```
Update customer
```bash
[PUT] http://localhost:8081/customers/{customerId}
```
